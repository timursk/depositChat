# Чат-бот

## Установка
Чтобы запустить проект , введите:
```sh
git clone https://github.com/timursk/depositChat.git
cd depositChat
node index.js
```
Далее вводите команды. <br>
**P.S. добавил 20% вероятность неудачного запроса данных для имитации обработки ошибки**
